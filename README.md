# 🌐 WebClones

A collection of frontend website clones created to improve my frontend development skills.

Each project focuses on recreating real-world websites with responsive layouts, clean code, and pixel-accurate user interfaces using modern web technologies.

> **Disclaimer:** All trademarks, logos, and original designs belong to their respective owners. These projects are created for educational and portfolio purposes only.

## 🎯 Goals

* Practice frontend development
* Improve HTML, CSS, and JavaScript skills
* Build responsive layouts
* Recreate real-world user interfaces
* Continuously improve UI implementation techniques

## 🛠️ Technologies

* HTML5
* CSS3
* Vanilla JavaScript
* Flexbox
