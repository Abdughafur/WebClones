# Google Company Info Clone

A clean and responsive clone of the **Google Company Info** page, built with **HTML5** and **CSS3**. This project recreates Google's modern design system while focusing on semantic markup, reusable components, and responsive layouts.

This clone is part of my **WebClones** collection, a repository dedicated to recreating modern websites for learning and frontend development practice.

## 🌐 Original Website

The original page can be found here:

**Google Company Info**
[https://about.google/company-info/](https://about.google/company-info/?utm_source=chatgpt.com)
